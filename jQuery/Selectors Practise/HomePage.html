Selectors Practise (All Can Click Show Effects)

$("*")

Select All Elements; Example: $("*")

$("#id")

Select Id Elements; Example: $("#textstyle1")

$(".class")

Select Class Elements; Example: $(".class")

$("element")

Select Elements; Example: $("h1")

$(":first")

Select Elements first Element; Example: $("h1:first")

$(":last")

Select Elements last Element; Example: $("span:last")

$(":even")

Select Elements even Element; Example: $("span:even")

$(":odd")

Select Elements even Element; Example: $("span:odd")

$(":first-child")

All elements that are the first child of their parent; Example: $("span:first-child")

$(":last-child")

All elements that are the last child of their parent; Example: $("span:last-child")

$(":first-of-type")

Select every element that is the first element of its parent; Example: $("span:first-of-type")

$(":last-of-type")

Select every element that is the first element of its parent; Example: $("span:last-of-type")

$(":nth-child(n)")

All elements that are the n child of their parent; Example: $("p:nth-child(1)")

$(":nth-last-child(n)")

All elements that are the 2nd child of their parent, counting from the last child; Example: $("p:nth-last-child(2)")

$(":nth-of-type(n)")

All elements that are the 2nd element of their parent; Example: $("span:nth-of-type(1)")

$(":nth-last-of-type(n)")

All elements that are the 2nd element of their parent, counting from the last child; Example: $("span:nth-last-of-type(1)")

$(":only-child")

Select each element that is the only child of its parent; Example: $("span:only-child")

$(":only-of-type")

All elements that are the only child, of its type, of their parent; Example: $("only-of-type")

$("parent>child")

All parent elements that are a direct child of a parent element; Example: $("#textstyle1>p")

$("parent child")

All elements that are descendants of a element; Example: $("#textstyle1 p")

$("element + element")

The element that are next to each elements; Example: $(".textstyle + p")

$("element ~ element")

All elements that appear after the element; Example: $(".textstyle1 ~ div")

$("parent child:eq(n)")

The element in a list (index starts at 0); Example: $(".textstyle p : eq(1)")

$("parent child:gt(0)")

List elements with an index greater than 0; Example: $(".textstyle p : gt(0)")

$("parent child:lt(1)")

List elements with an index lower than 1; Example: $(".textstyle p : lt(1)")
